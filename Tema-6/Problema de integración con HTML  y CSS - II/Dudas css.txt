Fede, sí ningún problema, lo podés hacer con tablas completamente. Creo que es la forma más sencilla.
Por las dudas, tratá de no poner nombres de clases iguales a los nombres de los elementos HTML.
Como este caso: class="table"

No veo problemas con el CSS que planteaste.

Yo te recomendaría no plantear elementos <br> como en el segundo código.
Sino usar tablas anidadas:

<table> <-- tabla principal
   <tr> <-- primera fila
     <td>
       <table> <-- aquí dentro armas la mini tablita que tiene los valores de compra, la fecha de actualización. Etc
       </table>
     </td>
   ...
</table>
